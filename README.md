# AIFFEL_quest_rs

아이펠 리서치 코스 과제 저장소입니다. 모든 프로젝트 폴더에는 피어 리뷰를 위한 **PRT(Peer Review Template)**가 포함되어 있습니다.

## 🏗️ 디렉토리 구조
```bash
AIFFEL_quest_rs
├── MainQuest
│   ├── Quest01 ~ Quest05
├── Exploration
│   ├── Ex01 ~ Ex08
└── GoingDeeper
    ├── G01 ~ G09
```

## 🛠️ 작업 루틴
1. 해당 폴더의 `.ipynb` 파일에서 과제 수행
2. 완료 후 `git add . / commit / push` 진행
3. 동료의 리뷰를 위해 `README.md`(PRT) 확인
